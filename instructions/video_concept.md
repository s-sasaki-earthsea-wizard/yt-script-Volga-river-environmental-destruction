# 動画コンセプト

このシリーズ全体に適用される制作方針です。

## 基本フォーマット

- **尺**: 3分以内
- **構成**: 1動画1テーマ
- **スタイル**: 短く、濃く、わかりやすく

## 核心メッセージ

> アウトプットをすることはより良いインプットにつながる

YouTubeを始めたらインプットの仕方が劇的に変わった。これがシリーズを通じて伝えたい最も重要な主張。

## リサーチ方針

### 情報源

| カテゴリ | 情報源 | 特徴 |
| --- | --- | --- |
| 研究者向け | arXiv, GitHub | 最新研究、実装、トレンド |
| 一般向け | MIT Tech Review, 日経サイエンス | 噛み砕かれた科学記事 |
| 基礎 | 教科書, サーベイ記事 | 基礎の再発見、体系的理解 |

### 深掘りプロセス

1. ネタになりそうな情報を発見
2. AIエージェントを活用して深掘り調査
3. YouTube先行動画を調査（独自色を出せるか確認）
4. Goサイン or お蔵入り判断

### 参入障壁を下げるマインドセット

- 難しい論文を読む必要は必ずしもない
- 中学校や高校の宿題でも、模範解答と違う変わった解き方を思いついたなら十分投稿する価値がある
- 誰かの勉強の役に立つかもしれない

## 差別化の基準

- YouTube先行動画調査で独自色を確認
- 被っていたらお蔵入りも選択肢
- 「自分だから言えること」を大切に
